東方萃夢想 Alt Lobby System 東方萃夢想ロビーシステム(RC4) 操作説明

はじめに

このドキュメントは、東方萃夢想 Alt Lobby Systemのテスト版の操作説明を簡略にまとめたものです。

アプリケーションと同様に本ドキュメントもテスト版であり、機能の説明やスクリーンショットなどは予告なく変更されることがあります。

RC4に対応していますが、一部b10から変更のない項目については、古いドキュメントのままになっている箇所があります。

Alt Lobby Systemについて

Alt Lobby Systemは、同人サークル「黄昏フロンティア」様と「上海アリス幻樂団」様が製作された対戦型格闘ゲーム 「東方萃夢想 〜 Immaterial and Missing Power」(以下、東方萃夢想と略させていただきます)のネット対戦を支援するツール（いわゆるロビーシステム）です。

利用するには、東方萃夢想のほか、通信アプリケーション「th075Caster」（以下、Casterと略させていただきます）が必要です。

Casterをベースとした、「CowCaster」や 「RollCaster」を代わりに利用することもできます。レーティングシステムSuica!用に改造されたものも利用可能です。

類似機能を有するツールとして、th075 Lobby Systemがあります。

注意・免責

本アプリケーションは、現在テスト中です。予期せぬ不都合が発生することがあります。不都合によって生じた問題についての責任は負いかねます。

本アプリケーションが動かない問題があった場合でも、黄昏フロンティア様の掲示板ほか、他所に質問するのは絶対に止めて下さい。

Readmeに利用方法が記述されていますが、十分な説明がなされていないことがあります。

テスト版ということで、動作のチェックをしていただける方を募集しています。

以上の点を留意した上でご利用ください。

操作説明

接続ウィンドウ
起動時に表示される接続ウィンドウについての説明です。
設定ウィンドウ
接続時に使用するさまざまな設定値を変更する設定ウィンドウの説明です。
メインウィンドウ
サーバー接続時に、チャット(全体IM)やCasterを使った対戦を行うウィンドウの説明です。
